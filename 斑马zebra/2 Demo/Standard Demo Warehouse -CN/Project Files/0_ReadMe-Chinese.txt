部署到PC的文件 - 快速简明指南
2017年5月23号--Andy Zhang
重要：请确保您在手持设备和电脑的控制台使用1.2.108或者更新版本；手持设备apk和Velocity版本需相同。


文件夹包括4个Velocity Console 工程和2个安装文件. 

文件名: 	Velocity_Console_1.2.108.msi
描述:	Install file for the 1.2.104 console. (Install first)

文件名: 	Speakeasy_Console_Addon_1.2.104.msi
描述: 	The Speakeasy add on install file (Install second)



文件名: 	1 - Migrate (Ivanti)-Chinese.zip
描述: 		Velocity中文工程文件，黑底绿字传统显示 demo (只设置主机配置文件和显示语言或者扫描scanner profile)


文件名: 	2 - Optimized (Ivanti)-Chinese.zip
描述: 		Velocity中文工程文件，黑底绿字传统显示 demo，增加个性化键盘和基于情景字段的自动选择


文件名: 	3 - Modernized (Ivanti)-Chinese.zip
描述: 		Velocity中文工程文件，完全支持触屏的现代化模式


Files Name: 	4 - Speakeasy (Ivanti)-Chinese.zip
描述:		Velocity中文工程文件，支持现代化演示的Speakeasy (Speech)语音解决方案



额外信息: 
您需要在Velocity Console导入这些zip工程文件，才能够查看到具体的配置信息等。
